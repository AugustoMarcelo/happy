  - [react-router-dom](https://github.com/ReactTraining/react-router)

  - [react-leaflet](https://react-leaflet.js.org/)

## :computer: How to run

  ```bash
    # clone repository
    git clone https://github.com/AugustoMarcelo/happy.git

    # enter web folder
    cd web

    # download dependencies
    yarn

    # running web project
    yarn start
  ```
